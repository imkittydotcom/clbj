# clbj
"command line blackjack"

play blackjack in your command line against a virtual dealer!

# to play:
python clbj.py
select mode and number of hands to play

# modes

## basic
- no betting
- dealer does not hit on any 17s

## casino
- betting
- 100 chip start
- dealer hits on soft 17s
